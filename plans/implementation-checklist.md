# FamilyLedger Implementation Plan (from Feishu doc)

## Phase 1: Tracer Bullet — 注册登录 + 记一笔账 + 同步
**Acceptance criteria:**
- [x] 邮箱注册 → 登录 → 获得 JWT
- [x] 首次登录自动创建默认账户
- [x] 预设分类已存在（餐饮、交通、工资等）
- [x] 记一笔支出 ≤ 3 步完成（金额 → 分类 → 确认）
- [ ] 断网时可记账，联网后自动同步
- [ ] 另一台设备登录可看到已同步的交易
- [x] 数字键盘有触感反馈，分类选择器有图标
- [ ] 深色/亮色主题均可用
- [x] Docker Compose 一键启动后端

### Phase 1 收尾任务:
1. 真实 gRPC 联调（注册/登录走服务端）
2. WebSocket 实时同步
3. 总余额显示修复
4. 深色模式验证
5. 触感反馈（HapticFeedback）

## Phase 2: 家庭协作 + 多账户 + 权限
**Acceptance criteria:**
- [ ] 创建家庭，生成 24 小时有效邀请码
- [ ] 他人通过邀请码加入
- [ ] 管理员可设置成员权限
- [ ] 普通成员受权限限制
- [ ] 顶部切换个人/家庭，带动画
- [ ] 添加多个资金账户
- [ ] 账户间转账不影响总额
- [ ] 家庭成员操作实时同步

## Phase 3: 预算管理 + 通知
- [ ] 设置月度总预算和分类子预算
- [ ] 记账后执行率实时更新
- [ ] 进度条颜色随执行率变化
- [ ] 超支时推送通知 + 脉冲动画
- [ ] 可自定义通知开关和提醒时间

## Phase 4: 贷款跟踪
- [ ] 等额本息/等额本金计算与银行一致（误差 ≤ 1 分）
- [ ] 查看未来每月还款明细（本金+利息分开）
- [ ] 提前还款模拟：显示节省利息和缩短月数
- [ ] 利率变动后剩余计划自动重算
- [ ] 还款日、信用卡账单日推送提醒
- [ ] 时间线视图滑动流畅

## Phase 5: 投资跟踪 + 实时行情
- [ ] 支持 A 股、港股、美股、加密货币
- [ ] 行情 15 分钟内刷新
- [ ] 迷你走势图嵌入列表
- [ ] 三种收益率计算正确（table-driven 测试）
- [ ] 图表触摸交互流畅
- [ ] 模块可扩展（新品种不改核心代码）

## Phase 6: 固定资产 + 折旧
- [ ] 添加房产/车辆/自定义资产
- [ ] 直线法 + 双倍余额递减法计算正确（单元测试）
- [ ] 车辆预设: 5 年、残值率 5%
- [ ] 每月自动折旧更新净值
- [ ] 手动更新估值后基数调整
- [ ] 估值历史折线图

## Phase 7: Dashboard + 报表 + 数据导出
- [ ] 净资产 = 账户余额 + 投资市值 + 固定资产净值 - 贷款余额
- [ ] 一屏展示关键指标
- [ ] 收支趋势: 月/年切换，触摸数据点
- [ ] 分类饼图: 点击扇区显示明细
- [ ] 卡片可拖拽排列、展开/折叠
- [ ] 导出 CSV / Excel / PDF 格式正确
- [ ] 支持按时间、分类筛选和全量导出

## Phase 8: 多币种 + CSV 导入 + OAuth 登录
- [ ] 记账可选币种，自动换算人民币
- [ ] 汇率每小时更新，API 不可用时降级用缓存
- [ ] CSV 导入: 上传 → 预览 → 映射 → 导入
- [ ] 微信 OAuth + Apple Sign In 登录成功
- [ ] 交易可添加备注、标签、图片
- [ ] Dashboard 统一以人民币展示

- [ ] 所有页面有空状态插图

- [ ] 错误状态有友好页面 + 重试
- [ ] 1000+ 条交易列表滚动流畅
- [ ] 深色/亮色模式视觉均正确
