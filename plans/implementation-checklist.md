## Phase 4b: 组合贷款增强

- [x] LPR 利率调整月可选（每年1月 / 放款对应月）

- [x] 列表页组合贷卡片：分段进度条 + 分拆月供展示
- [x] 向后兼容：现有独立贷款不受影响
